# 📖 About the Project

**VidyaMitra** is a modern **AI-driven web application** designed to improve digital learning and career development.

The system analyzes user resumes, evaluates skills, and provides personalized recommendations for learning paths, interview preparation, and career growth.

The platform uses a **full-stack architecture** with:

- **React.js** for a responsive frontend
- **FastAPI (Python)** for backend APIs
- **AI Agents** for intelligent decision making

# 📚 Use Case Scenarios

## 1️⃣ Resume Evaluation & Skill Mapping
A final-year engineering student uploads a resume.  
VidyaMitra analyzes the resume and identifies missing skills such as **data visualization or cloud computing** and recommends courses to improve employability.

---

## 2️⃣ AI Mock Interview
A graduate uses the **AI Interview Simulator** to practice interviews.  
The system analyzes **confidence, communication, and answer accuracy** and provides feedback.

---

## 3️⃣ Career Path Recommendation
A working professional planning to transition into **Data Science** uploads their resume.  
VidyaMitra suggests **certifications, courses, and learning paths** for a smooth transition.
